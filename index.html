GISHydroWEB

GISHydroWEB is a GIS-based web application for performing hydrologic analysis developed by The University of Maryland in conjunction with MDOT State Highway Administration (SHA). It consists of a database of hydrologic layers and customized modeling tools.

Some Features Include:

  • Complete database of DEM, Land use, and Soils data for Maryland drainage areas (includes parts of DE, PA, VA, and WV).
  • Watershed delineation.
  • Watershed statistics (drainage area, CN, Tc, and more).
  • Peak discharge estimates from Fixed Region Regression Equations (with confidence intervals and gauge calibration).
  • Built-in WinTR-20 model.
  • Download Layers for external use
  • Upload user-defined watersheds

The application is in a "Beta" testing phase, things may not work as expected. We Recommend using Chrome or Firefox. Please report any bug you encounter.

To start go to the Project Set-up tab

GISHydroWEB is under continuous revision. Although we are not a web application development company and do not charge for the use of our website, we do try to support it to the best of our ability. Subwatershed

Subdivide the watershed into subwatersheds to create stream reaches that will define the stream network topology for WinTR-20 modeling (optional). After clicking Yes the option to add flow paths or outlets will become available. Otherwise, the Time of Concentration tab will be enabled, this could take a couple of seconds while GISHydroWEB gathers all the required information to continue the analysis.

Subdivide

Flow paths and outlets will divide the watershed in subwatersheds. At least one flow path must be selected. Outlets must be placed on valid streams. After this process has finished you can download the subwatersheds in JSON format or proceed to the Download tab to e-mail it as a package (not working in beta test!!!!).

Time of Concentration

Select a time of concentration method and options, then click Apply and wait for the process to finish. This could take some time depending on the amount and size of the subwatershed(s). Tapping on a subwatershed will show up the respective ID# (available after Tc estimate). If the Velocity Method is selected, clicking on Show Tc will allow the user to merge by segment and recompute the Tc values in a new window. Otherwise, the Show Tc button will only show the Tc values for each subwatershed.
